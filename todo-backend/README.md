# Todo Backend

Todo Backend API 서버

## 시작하기

### 필요 조건

- Node.js (v18 이상 권장)

### 설치

```bash
npm install
```

### 실행

```bash
# 개발 모드 (파일 변경 시 자동 재시작)
npm run dev

# 프로덕션 모드
npm start
```

## 프로젝트 구조

```
todo-backend/
├── src/
│   └── index.js      # 메인 엔트리 포인트
├── package.json      # 프로젝트 설정 및 의존성
└── README.md         # 프로젝트 문서
```

## Express.js 사용하기

Express.js를 사용하려면 다음 명령어로 설치하세요:

```bash
npm install express
```

그리고 `src/index.js` 파일의 주석을 제거하면 기본 Express 서버가 동작합니다.
