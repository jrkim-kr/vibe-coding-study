# Todo React

Vite를 사용한 React Todo 애플리케이션입니다.

## 시작하기

### 환경변수 설정

`.env.example` 파일을 참고하여 `.env` 파일을 생성하고 백엔드 API 주소를 설정하세요.

```bash
cp .env.example .env
```

`.env` 파일에 다음 내용을 추가하세요:

```
VITE_API_BASE_URL=https://your-backend-url-here.com
```

### 의존성 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

### 빌드 미리보기

```bash
npm run preview
```
